- Git 상태 clean, main 브랜치 기준 작업

### 2. 일기 저장 기능

- 필드 구성: uid, content, createdAt
- 익명 로그인 사용자 기준 데이터 분리 저장

- 일기 0건일 때 안내 문구 표시

- Save 버튼 로딩 표시

### 6. 삭제 기능 추가
- 개별 일기 삭제(deleteDoc) 구현
- 사용자 확인(confirm) 후 즉시 반영
